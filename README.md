# blog_web


## 功能介绍

|        | 功能                                                     |
| ------ | -------------------------------------------------------- |
| 用户   | 用户管理：实现用户的注册登录、信息修改                   |
|        | 文章浏览：文章搜索、文章收藏、文章评论                   |
|        | 博客管理：写文章、修改文章                               |
| 管理员 | 用户管理：实现管理员的注册登录、信息修改，增删用户账户   |
|        | 文章浏览：文章搜索、根据用户ID筛选文章评论、查看文章日期 |
|        | 博客管理：增添文章 ，删除文章                            |
